Bugzilla is a massive system used by everyone from Mozilla to NASA. It does absolutely everything. There's only one problem — it sucks.

After working at Mozilla for a few months, I began development of an add-on that would add features to Bugzilla and make it nicer to use. Features include keyboard shortcuts, inline images and github commits, gravatars, relative timestamps, links for easier navigation and a competely new UI for comments.

The add-on is used by over half of Mozilla, along with hundreds of other Bugzilla users.
